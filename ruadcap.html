Urgent Action Recommended

Rua/Kivu Noir’s 1 TB priority data cap is insufficient following a network extension that increased the number of users, driving usage from 580 GB (Mar 13–Apr 12) to a projected ~1.9 TB (Apr 12–May 11). At the current rate of ~55 GB/day, the cap is exhausted in ~19 days of a 30-day cycle, forcing reliance on Standard data for ~11–12 days. Starlink’s planned reduction in Standard data speeds could disrupt operations for the expanded user base. An immediate increase to a 2 TB priority data cap is recommended to ensure reliable connectivity. Consider Peplink solutions to optimize bandwidth for high-demand scenarios, as seen in similar deployments.

Standard Data Risks

When priority data is exhausted, Rua/Kivu Noir is bumped to Standard data, which is deprioritized and subject to slower speeds, especially during peak usage times. With Starlink’s upcoming changes, Standard data speeds may drop significantly, impacting the increased user base from the network extension. Upgrading to a 2 TB priority cap mitigates this risk, ensuring consistent performance for all users.
